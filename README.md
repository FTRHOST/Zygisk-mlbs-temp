# Android Mod Menu Template (ImGui + Zygisk) (MLBS)

![GitHub](https://img.shields.io/github/license/LGLTeam/Android-Mod-Menu?style=flat-square)

Floating mod menu template for Android games (IL2CPP) using ImGui, Zygisk, and hooking. This repository provides a comprehensive base for creating game mods with advanced features like **Room Info**, **Battle Stats**, and a built-in **Web Server**.

## Features Overview
- **Floating Menu:** ImGui-based overlay for easy interaction.
- **Zygisk Module:** Clean injection method using Magisk/KernelSU.
- **Memory Hacking:** Example implementation of reading game memory (IL2CPP).
- **Web Server:** Expose game data (stats, player info) via local HTTP API.
- **Room Info:** View detailed opponent information.
